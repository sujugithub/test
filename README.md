# Project Title
*A 2D RPG built with Godot*

## About
This is a 2D role-playing game where players explore, interact with characters, complete quests, and battle enemies. The project is being developed as part of my game development and programming portfolio.

## Features
- Top-down movement system
- NPC interaction dialogue
- Quest system
- Inventory system
- Combat mechanics
- Save / Load system *(if added)*

## Built With
- Godot Engine
- GDScript
- Pixel Art Assets

## Goals
- Improve game development skills
- Learn game architecture & logic design
- Build a playable portfolio project
